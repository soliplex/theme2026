# Theme Template

A white-label reference app and starting template for building an AI chat
application on [soliplex_frontend](https://github.com/soliplex/frontend).

It is a thin wrapper around the library: all core functionality (chat,
history, settings, auth) comes from `soliplex_frontend`. This repo only
provides branding, configuration, and per-platform scaffolding.

## Setup

See [docs/app-setup.md](docs/app-setup.md) for build and run instructions.
